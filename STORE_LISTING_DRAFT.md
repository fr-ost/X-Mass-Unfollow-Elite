# Chrome Web Store Listing Draft

## Extension name
X Mass Unfollow Elite

## Short description
Controlled X/Twitter unfollow manager with delay, limits, cooldown, pause/resume, and clean UI.

## Detailed description
X Mass Unfollow Elite helps users manage their X/Twitter following list with controlled unfollow sessions.

Features:
- Unfollow non-followers
- Unfollow all visible following accounts
- Pause, resume, and stop controls
- Random delay between actions
- Max unfollow limit per session
- Cooldown after selected number of actions
- Skip verified/private-looking accounts
- Clean professional popup and settings page

Developed by Shahriar Ahmed.
Support/contact: Telegram @igfrostt

Important:
This extension is intended for personal account management. Users are responsible for complying with X/Twitter rules and platform limits.

## Category
Productivity

## Single purpose
The extension provides a controlled interface for managing unfollow actions on the user’s own X/Twitter following page.

## Permission justification
activeTab: Used to communicate with the currently open X/Twitter tab when the user clicks extension buttons.
storage: Used to save user settings such as delay, limits, cooldown, and behavior preferences.
host permissions for x.com/twitter.com: Required so the content script can run only on X/Twitter pages and interact with the following list.

## Privacy practices
The extension does not collect, sell, transmit, or share personal data.
Settings are stored locally using Chrome storage.
No external analytics or tracking is included.
